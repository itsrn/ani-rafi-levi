- `episode.json`: The episode and season number of every sentence
- `first_part.json`: The first part of every sentence. (The X in `×× ×™ ×¨×¤×™ ×œ×•×™. ×‘×™×œ×“×•×ª×™, ×œ× ×”×™×” ×œ× ×• X`).
